Linux命令备忘录

基础 Linux
基础
发布日期:   2020-11-28

man

man command
查看命令相关用户手册

init

init 0 关机
init 1 单用户模式
init 2 多用户模式，没有nfs，不联网，其他和3相同
init 3 完全多用户模式
init 4 没有启用
init 5 图形化界面
init 6 重启

在CentOS系统下/etc/inittab可以设置默认init

ls

列出文件和目录
-a 显示隐藏文件
-l 显示详细信息
-R 显示子目录
-h 带单位显示文件大小

tree

直观的显示多级目录

cat

cat example 查看example文件内容
-n 带行号
-b 带行号，但是跳过空白行

more

more example查看文件example
一页显示不下会显示进度
方向键和空格按行滚动
page up/down翻页
斜杠/搜索，n下一个匹配的字符，类似于vim
q 退出

less

less example查看文件example
类似more，支持更多特性

head

head example 显示文件前十行
head -6 example 显示文件前六行

tail

tail example 显示文件最后十行
tail -6 example 显示文件最后六行
tail -f example 显示文件最后一行，实时更新

cut

cut 命令从文件的每一行剪切字节、字符和字段并将这些字节、字符和字段写至标准输出
-b 剪切字节
-n ：不分割多个字节的字符，如果字符的最后一个字节落在由 -b 标志的 List 参数指示的范围之内，该字符将被写出；否则，该字符将被排除。
cut -b 3 file
cut -b 1-2,3 file
cut -b -3,3- file
-c 字符
和-b一样
-f 域
-d 分隔符（默认制表符），空格-d ‘ ‘
cat /etc/passwd|head -n 5|cut -d : -f 1,3-5
root:0:0:root
bin:1:1:bin
daemon:2:2:daemon
adm:3:4:adm
lp:4:7:lp

sort

以默认的方式将文本文件的第一列以ASCII 码的次序排列，并将结果输出到标准输出
-b 忽略每行前面开始出的空格字符。
-d 排序时，处理英文字母、数字及空格字符外，忽略其他的字符。
-f 排序时，将小写字母视为大写字母。
-i 排序时，除了040至176之间的ASCII字符外，忽略其他的字符。
-n 依照数值的大小排序。
-u 意味着是唯一的(unique)，输出的结果是去完重了的。
-o<输出文件> 将排序后的结果存入指定的文件。
-r 以相反的顺序来排序。

uniq

去除重复的行，重复的行不相邻不起作用，所以sort先排序
uniq file1 file2
file1 输入文件
file2 输出文件，不指定就输出到标准输出
-c 去重后在行首显示重复次数
-d 仅显示重复的行
-u 显示只出现一次的行

seq

seq 1 5
1 2 3 4 5
产生一到十之间的整数
-s 指定字符串分隔数字
-w 在列前加0使其宽度相同

Tr

tr 命令用于转换或删除文件中的字符。
从标准输入设备读取数据，经过字符串转译后，将结果输出到标准输出设备
tr -c -d 字符集1 字符集2
-c：反选设定字符。也就是符合字符集1 的部份不做处理，不符合的剩余部份才进行转换
-d：删除指令字符
cat testfile |tr a-z A-Z
转换大小写

grep

cat 1.txt | grep aaa
显示1.txt中有aaa的行
grep -rn test ./
查找当前目录下的文件中有test的行，显示行号

find

find . -name “my*” -ls
搜索当前目录中，所有文件名以my开头的文件，并显示详细信息

locate

locate命令其实是“find -name”的另一种写法，但是要比后者快得多，原因在于它不搜索具体目录，而是搜索一个数据库（/var/lib/locatedb），这个数据库中含有本地所有文件信息。Linux系统自动创建这个数据库，并且每天自动更新一次，所以使用locate命令查不到最新变动过的文件。为了避免这种情况，可以在使用locate之前，先使用updatedb命令，手动更新数据库。
locate -i ~/m
搜索用户主目录下，所有以m开头的文件，并且忽略大小写

which

在环境变量中搜索，返回第一个，可以看是否存在命令，用的是哪一个

whereis

查找二进制文件（-b），man说明文件（-m），源代码文件（-s）

whatis

获取指令简介
查看命令所在man帮助信息的目录
该命令在系统启动大约七十分钟后生成whatis数据库
可以用makewhatis立刻初始化数据库

chmod

chmod -R [ugoa][+-=][rwxX] example
-R 递归修改子目录权限
u 表示该文件的拥有者
g 表示与该文件的拥有者属于同一个组(group)的人
o 表示该组以外的人
a 表示所有人
+ 表示增加权限
- 表示取消权限
= 表示唯一设定权限
r 读取
w 写入
x 执行

或者chmod 744 example

chown

chown -R user[:group] example

-R 递归修改
修改文件所有权为user，group组

ps

ps aux 显示所有进程

netstat

netstat -pantu 显示所有端口连接

ifconfig

查看网卡信息
ifconfig eth0 up/down
启动/关闭网卡
ifconfig eth0 127.0.0.1 up
改网卡IP

iwconfig

查看网卡无线详细信息

dmesg

硬件变化日志

macchanger

修改网卡mac地址
macchanger -m E0:DC:FF:E1:E9:A1 wlan0

History

history查看命令历史
-c 清空命令历史
-d 删除一条指定命令
-w 将历史保存到指定文件中
